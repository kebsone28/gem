Présentation du Système

Bienvenue dans l'interface de gestion de l'électrification de masse. Cet outil a été conçu pour centraliser et optimiser toutes les étapes du déploiement électrique.

Objectifs principaux :

  • Suivi en temps réel des raccordements
  • Optimisation de la logistique des équipes
  • Contrôle budgétaire et reporting précis
  • Support 100% Hors-Ligne (PWA) pour le terrain

Utilisez le Dashboard pour une vue d'ensemble, ou plongez dans les modules spécifiques pour des actions détaillées.

Module Terrain

Le module Terrain est le cœur opérationnel du système. Il permet de visualiser chaque ménage sur une carte interactive.

Import de Données

Vous pouvez importer des fichiers Excel (.xlsx) pour mettre à jour massivement la liste des ménages.

Statuts & Suivi

Chaque ménage suit le workflow : Non débuté → Murs → Réseau → Intérieur → Réception validée, ou Problème / Inéligible.

Import Intelligent : Lors d'un import Excel ou Kobo, le système effectue un Upsert. Il fusionne les nouvelles données (comme les coordonnées GPS) sans détruire l'historique d'évolution des statuts existants. La carte se met à jour en temps réel.

Astuce : Cliquez sur un marqueur sur la carte pour voir les détails d'un ménage et modifier son statut en un clic.

🗺️ Outils Carte Avancés

La carte dispose de 10 outils avancés accessibles via la barre d'outils latérale gauche : tournée optimisée, photos terrain, mesure, annotations, et plus. Consultez l'onglet Outils Carte Avancés pour le détail.

🚗 Itinéraire Multi-Mode

Cliquez sur un marqueur → 🗺️ Itinéraire pour tracer la route depuis votre position. Dans le panneau d'instructions, basculez entre :

  • 🚗 Voiture — itinéraire routier via OSRM
  • 🚶 À pied — itinéraire piéton (temps estimé à 5 km/h)

🖥️ Mode Plein Écran

Cliquez sur Plein écran pour étendre la carte sur toute la fenêtre. Appuyez sur Échap pour quitter.

Gestion des Charges

Ce module permet d'équilibrer la charge de travail entre les différentes équipes opérationnelles.

Répartition Automatique

Le bouton "Répartition Automatique" utilise un algorithme intelligent pour assigner les ménages en attente aux équipes disponibles en fonction de leur capacité journalière.

  • Maçons : 2 ménages/jour
  • Branchement Réseau : 10 ménages/jour
  • Installations Intérieures : 8 ménages/jour
  • Contrôle de conformité : 15 ménages/jour

Rapports & Analytics

Générez des rapports détaillés pour le suivi d'avancement et la justification des coûts.

Export PDF

Génère un document formel incluant les statistiques clés et le tableau des interventions filtrées.

Export Excel

Télécharge les données brutes filtrées pour une analyse personnalisée dans Microsoft Excel ou Google Sheets.

Outils Carte Avancés

La barre d'outils latérale gauche de la carte donne accès à 10 fonctionnalités avancées spécialement conçues pour le suivi terrain de l'électrification.

📅 1. Tournée du Jour (Optimisation)

Sélectionnez les ménages à visiter, puis cliquez 🚀 Optimiser. L'algorithme du plus proche voisin (TSP) calcule l'ordre optimal de visite pour minimiser les trajets. La route apparaît en pointillés orange sur la carte avec les numéros d'ordre.

📸 2. Infos Ménage & Photos

Chaque popup de ménage (clic sur un marqueur) vous montre désormais les équipes assignées directemement (icônes et métiers). Vous pouvez modifier le statut d'un ménage en temps réel, ou ajouter des photos géolocalisées via la fonction + Ajouter.

📊 3. Heatmap de Progression

Affiche une grille colorée (~5 km) sur la carte : ■ Vert = zone terminée (>80%), ■ Orange = en cours (50-80%), ■ Rouge = en retard (<20%). Cliquez sur une zone pour voir le détail (X/Y terminés).

📏 4. Outil de Mesure

Activez le mode mesure (curseur en croix). Cliquez pour ajouter des points de mesure — la distance cumulative s'affiche en temps réel. Double-cliquez pour terminer. Idéal pour estimer les longueurs de câblage ou les distances entre sites.

✏️ 5. Annotations / Notes

Cliquez sur la carte pour placer une note textuelle (étiquette violette 📝). Les annotations sont sauvegardées automatiquement dans le navigateur et persistent entre les sessions. Utile pour marquer des obstacles, rivières, ou remarques terrain.

🔔 6. Alertes de Proximité

Activez pour recevoir une notification automatique quand vous êtes à moins de 500m d'un ménage non traité. Un cercle rouge de 500m s'affiche autour de votre position. Un signal sonore vous prévient. Nécessite la géolocalisation activée.

📴 7. Application PWA Hors-Ligne

L'application complète ("Service Worker") s'installe sur votre appareil et fonctionne sans connexion internet. Vos modifications se synchronisent automatiquement une fois le réseau retrouvé, grâce à la stratégie "Stale-While-Revalidate".

⏱️ 8. Timeline / Historique

Affiche un slider temporel en bas de la carte (30 derniers jours). Déplacez le curseur pour visualiser la progression dans le temps — les marqueurs s'estompent progressivement pour simuler l'état passé du projet.

🎯 9. Assignation Intelligente

Répartit automatiquement les ménages en attente entre X équipes. L'algorithme K-Means regroupe les ménages par proximité géographique. Chaque équipe reçoit un nombre défini de ménages avec un code couleur distinct sur la carte.

📄 10. Rapport PDF Cartographique

Génère un rapport imprimable dans un nouvel onglet avec : KPIs principaux (total ménages, statuts, terminés), tableau de répartition par statut avec pourcentages. Utilisez Ctrl+P pour sauvegarder en PDF.

Astuce : Tous ces outils sont accessibles depuis la barre d'outils verticale à gauche de la carte. Survolez chaque bouton pour voir son nom. Les boutons actifs changent de couleur.

Module Logistique & Analyse Prédictive IA

Le module Logistique centralise le suivi de l'atelier de précâblage, des livraisons par agent, du stock de matériel, et intègre désormais une couche IA prédictive pour estimer la date de fin de chantier.

Onglet Atelier & Chargement

Cet onglet vous permet de saisir chaque jour le nombre de kits précâblés et de kits chargés par région :

  1. Sélectionnez la région concernée dans le menu déroulant.
  2. Saisissez le nombre de Kits Précâblés (kits assemblés à l'atelier) et de Kits Chargés (kits montés sur véhicule pour livraison).
  3. Cliquez sur Enregistrer le rapport. Les totaux globaux et régionaux sont mis à jour immédiatement.

Historique automatique

À chaque enregistrement, un snapshot journalier est sauvegardé dans la base de données locale (30 jours retenus). C'est cet historique qui alimente le moteur IA prédictif.

Analyse Prédictive & IA

Le panneau "Analyse Prédictive & IA" en haut de l'onglet Atelier affiche 3 indicateurs calculés automatiquement :

Cadence / jour

Moyenne des kits précâblés par jour sur les 7 derniers jours d'activité. Affiche -- kits/j si moins de 2 jours d'historique sont disponibles.

Fin Estimée

Date calculée en divisant les kits restants par la cadence moyenne. Se met à jour automatiquement à chaque rapport journalier.

État du projet

Badge coloré : En avance (dans les délais), À surveiller (< 7j de retard), En retard (plus de 7j de retard).

Astuce : Pour voir la prédiction fonctionner immédiatement, enregistrez un premier rapport aujourd'hui, puis revenez demain et enregistrez un second. Le système calculera automatiquement la tendance.

Autres onglets Logistique

📦 Stock Matériel

Tableau de tous les ménages avec leur inventaire matériel (câbles, coffrets, disjoncteurs...). Filtrable et exportable en CSV.

🚚 Livraisons

Historique des livraisons par agent. Filtrable par date, agent, et statut de validation.

👷 Agents

Performance des agents : nombre de visites, temps moyen, date d'activité, et indicateur d'activité (Actif / Ralenti / Inactif).

⚠️ Anomalies

Détection automatique des alertes : agents inactifs depuis plus de 3 jours, visites de plus de 2h, ménages avancés sans matériel.

Grappes & Sectorisation

Regroupement géographique des ménages par proximité GPS (K-Means). Une Grappe représente une zone large, tandis qu'une Sous-Grappe correspond au périmètre d'action d'une équipe d'installation intérieure spécifique (~100 ménages).

Comment synchroniser avec KoboCollect ?

Allez dans le module Terrain ou utilisez le gestionnaire. L'importation est désormais "intelligente" et préserve tout votre historique de statut et d'équipes assignées. La carte sera mise à jour instantanément à l'écran sans avoir besoin de recharger.

Comment restaurer une sauvegarde ?

Utilisez le bouton Restaurer dans les paramètres du Module Terrain. Sélectionnez votre fichier JSON de sauvegarde. Attention : cela écrasera les données actuelles si elles sont plus anciennes.

Puis-je travailler sans internet sur le terrain ?

Oui. L'application est une PWA Hors-Ligne. Vos changements sont stockés localement grâce à la Data Index V2. La synchronisation PWA mettra tout à jour automatiquement en arrière-plan (Service Worker) dès le retour du réseau.

Comment suivre l'intelligence opérationnelle (Coûts & Productivité) ?

Le Dashboard analyse vos données pour vous fournir en temps réel le calcul de productivité et de l'avancement budgétaire estimé (grâce au CostCalculationService). Vous pouvez identifier les baisses de régime et anticiper les retards depuis cet écran central.

Pourquoi la cadence affiche « -- kits/j » ?

Le moteur IA a besoin d'au moins 2 jours d'historique pour calculer une tendance. Enregistrez un rapport aujourd'hui dans l'onglet Atelier & Chargement, puis revenez demain pour un second. La cadence et la date estimée s'afficheront automatiquement.

Comment sont générées les grappes géographique ?

Le système utilise un algorithme de clustering K-Means qui analyse les coordonnées GPS. Il crée des zones denses pour minimiser les déplacements des équipes. L'admin peut ensuite réaffecter manuellement une équipe spécifique à une sous-grappe via l'onglet Grappes du module Logistique.

Comment suivre la production par région ?

Dans l'onglet Atelier & Chargement, sélectionnez la région dans le menu déroulant avant de saisir les kits. Le tableau régional en bas de page se met à jour pour montrer les totaux et le taux de chargement par région, et les chiffres globaux s'agrègent automatiquement.
